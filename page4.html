Understanding Comments and Attributes in HTML

Comments

Comments are messages written by developers to better understand or communicate ideas with others in their code. Comments do not appear on your webpage; however, they do appear in your code.

To add a comment in HTML, use the following:

<!-- This is a comment -->

Example:

This is what the code would look like...

<!-- This is a comment explaining something -->
<p>This text will be displayed instead.</p>

Resulting in:

This text will be displayed.

Attributes

Attributes provide additional information about an element. They are always specified in the opening tag and written as name="value".

Some common attributes include:

  • href: Specifies the URL for links.
  • src: Specifies the source for images or other media.
  • alt: Provides alternative text for images.
  • style: Adds inline styles to elements.

Examples:

Hyperlink with href attribute:

Code (Using the paper.io website):

<a href="https://paper-io.com/">Visit Example</a>

Resulting in:

Visit Example
