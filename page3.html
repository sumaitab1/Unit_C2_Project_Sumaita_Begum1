Formatting Tags

HTML has two types of body elements: block-level elements and inline elements. Block-level elements will automatically appear on separate lines in your code, while inline elements appear horizontally so that the text does not start a new line. The following tags are examples of inline elements!

  • <b>this tag makes text appear bold </b>
  • <strong>this tag is similar to the bold tag and also makes text appear bold </strong>
  • <i> this tag makes text appear italicized </i>
  • <em> this tag also makes text appear italicized </em>
  • <mark> With this tag, text will appear highlighted </mark>
  • <small> This tag makes text appear small </small>
  • <del> With this tag, text will appear crossed out </del>
  • <ins> This tag makes text appear underlined </ins>
  • <sub> This tag makes text appear as a subscript </sub> For example, H2O
  • <sup> This tag makes text appear as a superscript </sup>

Structural Tags

There are also list elements (block level elements) which consist of:

  • <ol> This tag creates an ordered list with numbers. However, the tag <ul> creates an unordered list with bullet points instead.

Note: Remember to always put <ul> or <ol> first, followed by the <li> tag, or else your code will appear incorrect when passed through a validator and the bullet points/numbers you wanted will not appear.

If you are curious about what an unordered list would look like, look above! The list of formatting tags shows different tags using the <ul> tag. Below is an example of an ordered list.

  1. This
  2. Is
  3. An
  4. Ordered
  5. List

The code would look something like the following...

<ol>
<li> Hello </li>
<li> World! </li>
</ol>

or...

<ul>
<li> Hello </li>
<li> World! </li>
</ul>
