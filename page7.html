Creating Tables

  1. Tables allow web developers to organize and arrange their data collected into a grid-like structure. For tables, we use the following tags:
    • <table> (This tag creates the actual table)
    • <tr> (This tag creates the row)
    • <th> (This tag creates the heading cell)
    • <td> (This tag creates a data cell)
  2. To create borders in HTML5, you will have to use CSS.
  3. In order to change the table and cell sizes, we use the following:
    • table {insert width and height in pixels}
    • For cell sizes: th, td {insert width and height in pixels}
    • For colspan and rowspan, we use <th colspan="#"> or <th rowspan="#">

Here is an example of what a table would look like:

Column #1 Column #2 Column #3 Column #4
Row Cell #1a Row Cell #1b Row Cell #1c Row Cell #1d

NOTE: Don't forget to put the style code to choose the thickness of your lines and color. I like pink, so the example above I used pink!

Here is an example of what the code for a table similar to the example would look like:

<table>
<tr>
<th>Column #1</th>
<th>Column #2</th>
<th>Column #3</th>
<th>Column #4</th>
</tr>
<tr>
<td>Row Cell #1</td>
<td>Row Cell #2</td>
<td>Row Cell #3</td>
<td>Row Cell #4</td>
</tr>
</table>
