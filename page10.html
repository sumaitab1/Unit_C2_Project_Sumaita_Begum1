HTML Entities

Several characters are reserved only for the use of HTML. However, other characters difficult to type on the keyboard, such as emojis, cannot be made without a special application. To display these characters, you can use the following methods:

  1. (Nickname method) & + nickname + ;
  2. (Number Method): & + # + Unicode Number + ;

Example: < becomes &lt; (without spaces)
