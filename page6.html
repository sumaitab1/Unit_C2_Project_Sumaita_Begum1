HTML Links and Images

Use the following examples to help you include a link to your code. Each example is created based off the type of link used from basic to downloadable all the way to images and more!

  1. Basic Link Syntax:

    A basic link creates a clickable text or element that navigates to a specified location.

    <a href="pathToFile">Link Text</a>

    Example:

    Visit Local File

  2. Absolute Link (external):

    An absolute link directs users to a specific webpage on the internet.

    <a href="https://www.google.com">Google</a>

    Example:

    Google

  3. Relative/Internal Link:

    A relative link connects to a file within the same website or a folder.

    <a href="/index.html">Home Page</a>

    Example:

    Home Page

  4. Downloadable Link:

    A downloadable link lets users download a file and insert it into their webpage instead of opening it in the browser.

    <a href="example-file.txt" download="example-file">Download File</a>

    Example:

    Download File

Images

  1. Basic Image Syntax:

    Images are added to webpages using the img tag, which requires the src attribute to specify the image source and the alt attribute for accessibility.

    <img src="example.jpg" alt="Description of your image">

    Example:

  2. Image Accessibility:

    The alt attribute provides a description for the image, making it accessible to visually impaired users by allowing screen readers to describe the image content.

  3. Adjusting Image Size:

    Use the width and height attributes to control the size of an image. Specifying both ensures consistent scaling.

    <img src="File" width="#" height="#">
