What is HTML?

HTML, also known as Hyper Text Markup Language, is the backbone of the World Wide Web. Created by Tim Berners-Lee in 1989, it is a language that helps format texts. In HTML, we use angle brackets as symbols to start and end our tags.

What do we need to start an HTML webpage?

  1. A doctype declaration
  2. An HTML tag beginning and ending the page
  3. A head tag that holds the page's metadata (learn about metadata in lesson #)
  4. Title Tag; shows up on web browser Tabs
  5. A body tag
Here is what the code would look like: 
      <!DOCTYPE html>
      <html>
      <head>
      <title> Title of Web Page </title>
      <body> (This is the tag to start your code) </body>
      </html>
