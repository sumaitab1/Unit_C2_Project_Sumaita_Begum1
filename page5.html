Headings and Paragraphs

Headings are used to create an outline structure on a web page. There are only 6 heading types from h1 to h6. For a heading tag, we use: <h#>

Paragraph tags are the workhorse tags of web design. Most blocks of text use paragraphs for formatting, so remembering this tag is crucial. For a paragraph tag, we use <p>. However, if you would like to include a line break, use the tag <br> to start a line break.

If you would like a line between the tags, use the tag <hr>, which is essentially just an empty tag. If you would like to preserve multiple spaces or line breaks, you can use the preformatted text <pre> tag. This tag is a block-level element.

  • Here is an example of how a heading tag would look in your code: <h#> Hello World! </h#>
  • However, a paragraph tag would look more like this: <p> This is an example of a paragraph tag </p>
